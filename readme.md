💊 MedSafe AI

MedSafe AI is an intelligent healthcare safety assistant designed to help users understand medicine interactions, analyze prescriptions, and receive educational health guidance. The system integrates OCR technology, rule-based safety logic, and AI-assisted modules to provide structured insights about medicines and potential health risks.

---

🚀 Features

- 🔍 Medicine Interaction Checker
  Detects potentially dangerous drug combinations using a medicine interaction database.

- 📷 Prescription OCR Scanner
  Extracts medicine names from prescription images using Tesseract OCR.

- 🧠 AI Structured Output
  Converts extracted prescription data into structured JSON format for transparent processing.

- 🤒 Symptom & Doubt Solver
  Provides educational guidance for common symptoms.

- ⚠ Side-Effect Monitor
  Displays known side effects for selected medicines.

- 🚨 Emergency Risk Predictor
  Estimates a risk score based on symptoms and medicine usage.

⚙ Installation

1. Clone the repository:

git clone https://github.com/yourusername/MedSafe-AI.git

2. Navigate to the project folder:

cd MedSafe-AI

3. Install dependencies:

pip install -r requirements.txt

4. Run the application:

streamlit run main.py

The application will open in your browser at:

http://localhost:8501
